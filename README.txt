http://go/neelix

Neelix is a versatile visual correlation tool for different kinds of
time-series metrics like Java Garbage Collection, CPU, I/O, Application
throughput etc.

Example Usage:

bin/neelix -i test/data/logs/ -o ~/tmp/neelix test/conf/config-sar 
View Results.html in firefox (Chrome complains about cross-domain issues)

bin/neelix -i test/data/logs/ -o ~/tmp/neelix test/conf/config-gc
